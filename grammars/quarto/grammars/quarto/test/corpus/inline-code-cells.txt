==================
Python inline code cell
==================

The result is `{python} 2 + 2`.

---

(document
  (blank_line)
  (paragraph
    content: (inline
      (text)
      (inline_code_cell
        (inline_cell_delimiter)
        language: (language_name)
        (inline_cell_brace)
        content: (cell_content)
        (inline_cell_delimiter))
      (text))))
